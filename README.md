A super simple dummy server for testing HTTP clients.
Zero dependencies.

## Installation

```bash
cargo install --git https://github.com/mre/teapot.git
```

## Usage

```bash
teapot
```

You can configure the port with the `PORT` environment variable.

```
PORT=8080 teapot
```
